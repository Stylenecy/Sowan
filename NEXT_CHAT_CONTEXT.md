## 2. PROBLEM STATEMENT

### 2.1 The Cultural-Language Learning Gap

When learning a language, most people master vocabulary and grammar. However, when it comes to actual interaction, they still feel hesitant. The real challenge isn't the language itself — it's understanding the **culture and social context** behind it.

**The Paradox:**
- Language learning apps (Duolingo, Babbel, etc.) are abundant and efficient
- AI translation tools are increasingly accurate
- Yet genuine cultural understanding remains elusive

### 2.2 The Elderly Isolation Problem

Indonesia has approximately **33.9 million elderly citizens** (ages 60+), representing about **12% of the population**. Many possess:

- Decades of professional experience
- Deep cultural and traditional knowledge
- Rich storytelling capabilities
- Patient, empathetic communication styles

**However:**
- Limited opportunities for economic participation
- Often feel isolated from modern society
- Skills and wisdom go underutilized
- "Old age" is stereotyped as "unproductive"

### 2.3 The Market Disconnect

| Global Need | Local Supply |
|-------------|--------------|
| Cultural integration support | 183,000+ foreign workers in Indonesia |
| Authentic cultural learning | Elderly wisdom largely untapped |
| Meaningful cross-generational connections | Limited platforms bridging this gap |

**The Result:** A massive untapped potential — elderly experience meets global cultural learning needs.

## 3. OUR SOLUTION: SOWAN.ID

### 3.1 What is SOWAN?

**SOWAN** (Indonesian word meaning "to visit" or "to seek wisdom from an elder") is an **edutech marketplace platform** that connects Indonesian youth and international learners with elderly mentors ("sesepuh") for meaningful 1-on-1 video call sessions.

**Core Value Proposition:**
> "Learning language through culture, from those who lived it."

### 3.2 Why Elderly Mentors?

Elderly mentors provide something **technology cannot replicate:**

1. **Contextual Learning** — Experience-based explanations that make language come alive
2. **Patient Guidance** — Unhurried, empathetic approach suited for anxious learners
3. **Authentic Storytelling** — Real life stories that create lasting impressions
4. **Cultural Depth** — Generational wisdom passed through conversation
5. **Emotional Connection** — Human warmth that AI/computers cannot match

### 3.3 Vision & Mission

**Vision:**
> Supporting global cross-cultural integration while contributing to SDGs 4, 8, and 10.

**Mission:**
> Creating meaningful economic opportunities for elderly while providing authentic cultural education to global learners.

## 4. BUSINESS MODEL

### 4.1 Revenue Model

**Commission-Based (30% per session)**

| Component | Details |
|-----------|---------|
| Platform Commission | 30% of session price |
| Mentor Earnings | 70% of session price |
| Session Pricing | Rp 66,000 — Rp 350,000 per hour |
| Target | 2,000 sessions per month by Year 2 |

### 4.2 Revenue Flow

```
Learner pays Rp 100,000 for session
        ↓
SOWAN receives full payment
        ↓
SOWAN keeps Rp 30,000 (30% commission)
        ↓
Mentor receives Rp 70,000 (70%)
```

### 4.3 Cost Structure

**Fixed Costs:**
- Platform development & maintenance
- Server infrastructure
- Customer support

**Variable Costs:**
- Payment gateway fees (~2-3%)
- Marketing & acquisition costs
- Mentor training programs

### 4.4 Growth Model

Business growth aligns directly with:
- Increased session volume
- Higher user retention rates
- Expansion of mentor network
- B2B partnerships (expat communities, educational institutions)

### 10.3 Data Flow

**Authentication:**
- User name stored in localStorage (`sowan_user`)
- Login modal sets user state
- Dashboard checks user type (customer vs mentor)

**Booking Flow:**
1. User selects mentor on explore page
2. User clicks "Konfirmasi Jadwal" → modal opens
3. User selects time slot
4. User confirms payment → success state
5. Booking data saved to localStorage:
   - `sowan_booked_mentor` — Mentor object
   - `sowan_selected_time` — Time slot string
   - `sowan_room_id` — Room ID

**Video Decision:**
```typescript
// Client-side logic in room/[id]/page.tsx
const maleMentorIds = [1, 3, 4, 6, 7, 10, 12, 13];

if (maleMentorIds.includes(mentorIdNum)) {
  // Show local MP4
  setVideoDecision({ isLocalVideo: true, videoId: '/video-pak-budi.mp4' });
} else {
  // Show YouTube embed
  setVideoDecision({ isLocalVideo: false, videoId: 'N90UIXMuMMU' });
}
```

### 10.4 LocalStorage Keys

| Key | Purpose | Data Type |
|-----|---------|-----------|
| `sowan_user` | Current logged-in user | `{ name: string, type: 'customer' \| 'mentor' }` |
| `sowan_booked_mentor` | Last booked mentor | Full mentor object |
| `sowan_selected_time` | Selected time slot | String (e.g., "Hari ini, 10:00 - 11:00 WIB") |
| `sowan_room_id` | Current room ID | String (mentor ID) |

## 13. KEY UI/UX DECISIONS

### 13.1 Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary | `#1A365D` | Headers, main text, CTA buttons |
| Accent | `#D97706` | Highlights, secondary buttons, stars |
| Background | `#FAF9F6` | Main background (warm off-white) |
| Success | `#22C55E` | Online status, success states |
| Warning | `#F59E0B` | Pending states |
| Error | `#EF4444` | Error states, destructive actions |

### 13.2 Typography

| Element | Font | Size | Weight |
|---------|------|------|--------|
| H1 (Headlines) | Playfair Display | 48-72px | Black (900) |
| H2 (Section Titles) | Inter | 24-32px | Bold (700) |
| Body | Inter | 16-18px | Medium (500) |
| Small Text | Inter | 12-14px | Regular (400) |
| Buttons | Inter | 16-20px | Bold (700) |

### 13.3 Component Patterns

**Cards:**
- Border radius: 24-48px (rounded-xl to rounded-[48px])
- Border: 1-2px with low opacity (`border-black/5` or `border-primary/20`)
- Shadow: Subtle shadows (`shadow-sm`, `shadow-xl`)
- Padding: 24-48px internal spacing

**Buttons:**
- Border radius: 16-28px (rounded-2xl to rounded-[28px])
- Height: 48-80px (h-12 to h-20 for CTAs)
- Shadow: Prominent for primary actions
- Hover: Scale up slightly (-translate-y-1)

**Modals:**
- Border radius: 48px
- Backdrop: Dark overlay with blur
- Animation: Fade in + scale up

### 13.4 Responsive Breakpoints

| Breakpoint | Width | Layout |
|------------|-------|--------|
| Mobile | < 640px | Single column, stacked |
| Tablet | 640-768px | 2-column grid |
| Desktop | 768px+ | Full layout with sidebars |

### 13.5 Accessibility (Geronteknologi)

**Mentor Dashboard Features:**
- Minimum font size: 18px
- Minimum button height: 56px
- Color contrast: 4.5:1 ratio
- Touch targets: 48px minimum
- Clear focus states
- High contrast borders
- Simple, uncluttered layouts
